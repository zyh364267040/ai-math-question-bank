# 高中数学高考知识点体系

## 一、体系用途

本体系用于 AI 数学题库的题目分类、标签管理、搜索选题、考试频率统计和后续智能组卷。

本体系不按教材册次和章节顺序划分，而是按照高考复习与解题能力链重组。每道题至少标注一个三级知识点；综合题可以同时标注多个知识点，并区分“主知识点”和“关联知识点”。

总体系：

> 基础工具 → 函数 → 三角函数与解三角形 → 平面向量 → 立体几何与空间向量 → 解析几何 → 数列 → 导数及其应用 → 概率统计 → 计数原理与二项式定理 → 综合与跨模块问题

## 二、题库标签使用规则

### 1. 主知识点

每道题必须指定一个最核心的三级知识点，作为主知识点。例如，一道利用导数研究参数范围的题，主知识点可以标为“08.06 导数与恒成立—分离参数”。

### 2. 关联知识点

一道题可以同时关联多个知识点。例如，圆锥曲线定值题可以关联“方程联立”“根与系数关系”“定值问题”。

### 3. 题型标签与知识点分开

知识点回答“考什么”，题型标签回答“怎么考”。同一个知识点可以对应选择题、填空题、解答题，也可以对应求值、证明、参数范围、最值、存在性等不同任务。

### 4. 高频数据不写进知识点名称

知识点出现频率由题库根据地区、年份、试卷和题号自动统计，不把“高频”“低频”固化在标签名称中。

### 5. 教材章节作为辅助字段

题库可以另外记录教材版本和教材章节，方便追溯知识来源，但不以教材章节作为题库的主分类体系。

## 三、还需结合样题继续校准的部分

- 天津卷中各三级知识点的实际覆盖情况
- 部分交叉题目的主知识点判定规则
- 难度分级标准
- 题型标签和解题方法标签体系
- 高频统计的年份范围和试卷范围
- 新定义、开放题和综合题的归类规则

本体系作为第一版分类骨架。后续录入真实天津高考题和模拟题时，应根据无法准确归类的题目持续补充和调整，而不是一开始追求完全固定。
